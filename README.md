Abstract
Understanding complex topics can be challenging for beginners and students. This project presents an AI-based learning assistant that simplifies complex textual content by explaining it in very simple language or generating the most commonly asked questions related to the topic.

The system accepts both text input and PDF documents, processes the content using Google Gemini Generative AI, and provides easy-to-understand explanations or relevant questions. The application is built using Python and Streamlit, offering a lightweight and interactive web interface.

OBJECTIVES

To simplify complex concepts using AI

To help beginners understand topics easily

To generate important and frequently asked questions

To allow learning directly from PDF documents

To demonstrate practical usage of Generative AI in education

Problem Statement

Students often struggle to understand complex academic material due to difficult terminology and lengthy explanations. Additionally, identifying important questions from study material requires manual effort.

This project aims to solve these problems by:

Converting complex text into child-friendly explanations

Automatically generating important questions

Supporting PDF-based learning
